TeamPass needs to be installed on a web server with next minimal configuration.

Server Requirements

  • Apache v2.0 (and after) OR Nginx
  • MariaDB 10 (and after) OR MySQL v8 (and after)
  • PHP v8.1 (recommanded 8.2)

 

 

Expected PHP modules

  • bcmath
  • curl
  • gd
  • gmp
  • iconv
  • LDAP (if using LDAP identification)
  • mbstring
  • mcrypt
  • mysqli
  • openssl
  • xml
